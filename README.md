#### turtle绘制图形
```Python
import turtle

turtle.pensize(4)
turtle.pencolor('red')
turtle.forward(100) #初始默认向右画
turtle.right(90) #右转
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.mainloop()
```
